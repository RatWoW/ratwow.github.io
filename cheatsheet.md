# Cheatsheet and references


## A quickguide for useful commands and reference links.  

| Command | Purpose | Pro-Tip |
| :--- | :--- | :--- |
| `.npc info` | Target Data | Get the **DisplayID** and **GUID** and more of an existing NPC. |
| `.gps` | Position Data | Get MapID, zoneid, **X, Y, Z, ** and **Orientation** for spawning. |
| `.npc add [id]` | Spawn Template | Quickly test the NPC template you just created. |
| `.npc delete` | Remove NPC | Deletes the *spawned instance* you are currently targeting. |
| `.lookup creature [name]` | ID Search | Finds the template ID for any NPC by its name. Use this to look at templates in Keira3.  |
| `.mod speed [#]`, `.gm fly on`  | Move Quickly | Use these to traverse the world swiftly
